# Student Registration Management System (C++)
A simple C++ based Student Registration System that allows adding, updating, deleting, searching, and viewing student details.  
The system also includes a login feature using a password, and supports club activity registration.

## Features
- Add new students
- Display all student records
- Search student by ID
- Update student details
- Delete student records
- Student login using ID + password
- Store club activities (comma-separated)
- Input validation for numeric fields

## Technologies Used
- C++ (Standard Library)
- File storage optional (data stored in runtime memory)
